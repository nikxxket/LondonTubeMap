🚇 Карта Лондонского метро — Интеллектуальный агент

Интерактивное Java-приложение, визуализирующее карту Лондонского метро с возможностью поиска кратчайшего пути и
симуляцией движения интеллектуального агента на основе вероятностной модели принятия решений.

Возможности:

🗺️ Визуализация карты:
1) Цветные линии метро (Красная, Желтая, Зеленая, Синяя);
2) Станции с пересадками (двойные круги);
3) Расстояния между станциями в километрах;
4) Интерактивная легенда.
   
🔍 Поиск кратчайшего пути:
1) Алгоритм Дейкстры;
2) Визуальное выделение маршрута фиолетовым цветом;
3) Отображение линий метро на каждом участке пути;
4) Расчет общего расстояния.

🤖 Интеллектуальный агент:
Принимает решения на основе полезности (utility) станций.

Вероятностная модель перемещения с учетом:
Расстояния между станциями, 
Направления движения (поощрение движения к цели), 
Случайных отклонений (10% вероятность исследования).

📊 Отображение данных:
1) Значения полезности (utility) для каждой станции;
2) Текущий статус агента (шаг, позиция, достижение цели);
3) Детальная информация о маршрутах.

🎮 Управление:
1) Выбор маршрута;
2) Выпадающие списки — выбор начальной и конечной станции;
3) Кнопка "Найти кратчайший путь" — расчет и отображение маршрута;
4) Запустить агента — инициализация агента с расчетом полезностей;
5) Шаг агента — выполнение одного шага симуляции;
6) Сбросить агента — очистка состояния агента.

🧠 Алгоритмы

Поиск кратчайшего пути (Дейкстра)

Классический алгоритм поиска оптимального маршрута по взвешенному графу.

Вероятностная модель агента

Агент использует итеративное обновление полезностей станций:

U(s) = R(s) + γ * max(Σ P(s→s') * U(s'))

где:

1) R(s) — награда за достижение цели или отрицательное расстояние до цели;
2) γ = 0.9 — коэффициент дисконтирования;
3) P(s→s') — вероятности перехода, взвешенные расстоянием и направлением.
